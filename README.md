# pyDentate

This python project is based on a dentate gyrus network model developed in the [Soltesz Lab](https://med.stanford.edu/ivansolteszlab.html).
This project is maintained by Daniel Müller at the [Institute of Experimental Epileptology and Cognition Research](http://www.meb.uni-bonn.de/agBeck/cms/front_content.php?idcat=333) Contact: danielmuellermsc@gmail.com
