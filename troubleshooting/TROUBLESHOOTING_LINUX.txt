# No module named neuron
Add the location of the neuron module to PYTHONPATH in the .bashrc
.bashrc is located in the home directory but it is hidden by default.
Type ls -la to find it
The neuron module was located by default in /usr/local/nrn/lib/python/
Add
export PYTHONPATH="/usr/local/nrn/lib/python/
to the .bashrc file
